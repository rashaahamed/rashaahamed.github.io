Getting Started with HTML & CSS


Posted 11/7/2021 Rasha Ahamed



To start coding with HTML, we must first create a file. When naming the file, it should always end with the suffix ‘.html’ so the program understands its file type.



The first characters on the document should always be the '!DOCTYPE' tag, which specifies an HTML document.



This line of code also lets the program know its file type.

But before you can start coding, you must understand the fundamental structure of HTML code.



What are HTML Elements?



To put it simply, an HTML element is just a line of code.

An element has an opening tag and a closing tag. These tags can hold other elements, or they can be empty. There are dozens of tags, but only a few are frequently used.



The ‘body’ tag is most often used. This tag is to define the main content of a website.





One of the most important tags is the ‘div’ tag. This tag is essentially a universal container. It can be used to hold text, an image, anything, or even nothing.

The div tag can be named too. This is done through an ‘attribute’.

HTML attributes are like sub tags; they define additional characteristics of an element. Attributes are always located in the opening tag of an element.

To name a div, you can use the ‘class’ or the ‘id’ attribute like so.



Block Level Vs Inline Elements



There are two types of elements, block and inline.

Block level elements display code in a new line, stacking one element after the other. Paragraph tags are set as block elements as default.



Inline elements display code on the same line. Span tags are set as inline elements as default.



Cascading Style Sheets



While HTML is used to structure a website, it is not enough to make a visually appealing website. With a separate coding language called Cascading Style Sheets (CSS), you can style HTML.

What is a CSS Rule?

A CSS rule is essentially a line of CSS code. There are three essential components in a CSS rule: the selector, property, and value.

Let’s say you wanted to make the body of an HTML document blue. You would first type ‘body’ as the selector, and then two curly brackets.

Inside the brackets is the content. To change the color of the background, you can do that with the ‘background-color’ property. To declare what color you want the selector as, type ‘blue’ in words, or find the precise hex code for it.
